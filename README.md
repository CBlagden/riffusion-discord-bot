# Riffusion Discord Bot

An easy-to-use discord bot which produces music from text prompts using [riffusion](https://www.riffusion.com/about).

## Setup

Follow the instructions to set up the [riffusion inference](https://github.com/riffusion/riffusion-inference) server first. You will need a GPU powerful to run the model for this.

Then clone this repo, and copy `example.config.json` to `config.json` and fill in your discord bot token, server guild id, and server url.
